Google Fonts
Custom fonts are used on this site. For ease of use, they are loaded from the Google Font service. The service will handle returning the correct font for the browser and I did not have add my own browser deletection logic. There are 2 additional benifits:
  • The site uses 2 fonts and Google Fonts allow me to specify in the URL which fonts I, and which font weights, and that I only need the latin characters. All of this is done to reduce the number of requests and the size of the response payload.
  • The reuqest URL has display=swap. This allows the responding CSS to contain font‑display: swap which is allows the text to render in the brower's default font while the Google font is being downloaded. A huge improvement in performance.
